## 关键架构细节（实施前必读）

### CausalMultiHeadSelfAttention 结构（来自 Assignment4 model.py）

`common/model.py` 的 `Transformer` 继承自 `BasicsTransformerLM`，其内部结构为：
- `self.layers`：`nn.ModuleList`，每个元素是 `TransformerBlock`
- 每个 `TransformerBlock` 有 `self.attn`（类型 `CausalMultiHeadSelfAttention`）
- `CausalMultiHeadSelfAttention.forward(x, token_positions=None)` 的关键代码：
  ```python
  # line 589-595 in reference Assignment4 model.py
  attn_output = F.scaled_dot_product_attention(
      query=Q,
      key=K,
      value=V,
      is_causal=True,
      enable_gqa=False
  )
  ```
  这是 Plan 1 留的注入 TODO 点。

### Attention 注入策略

**选择方式 A（猴子补丁）**，原因：
- `CausalMultiHeadSelfAttention` 将 `F.scaled_dot_product_attention` 调用封装在 `forward()` 中，结构清晰
- 猴子补丁只需替换该方法，不需要 override 整个 `Transformer.forward()`
- 方式 B（手动展开整个 forward）代码量大，维护难度高，YAGNI 原则下不选

**注入点**：在 `Transformer.__init__` 末尾，对 `self.layers` 中每个 block 的 `attn` 替换 `forward` 方法：

```python
# 在 Transformer.__init__ 末尾（super().__init__() 调用之后）添加：
self._attn_fn = _build_attn_fn(config.use_flash_attn)
_inject_attn_fn(self, self._attn_fn)
```

其中 `_build_attn_fn` 和 `_inject_attn_fn` 是模块级辅助函数（见 Task 1 详细代码）。

### `_build_attn_fn` 私有辅助函数（`common/model.py` 内部，非公共接口）

`_build_attn_fn(use_flash_attn: bool)` 是 `common/model.py` 的模块级私有辅助函数，
它委托调用 `backends/attention.py` 的公共接口 `get_attention_fn(use_flash_attn)`，
不重复实现 fallback 逻辑：

```python
# common/model.py（模块级私有辅助）
def _build_attn_fn(use_flash_attn: bool):
    """委托 backends/attention.py 获取当前最优 attention 函数。"""
    from llm_foundry.backends.attention import get_attention_fn
    return get_attention_fn(use_flash_attn)
```

`use_flash_attn=False`（CPU/开发环境，默认）：`get_attention_fn` 短路返回 F.sdpa，不探测硬件。
`use_flash_attn=True`（GPU 训练）：`get_attention_fn` 执行三级 fallback（Triton → torch.compile → F.sdpa）。

此函数不需要自行探测 `flash_attn` 原生库——探测逻辑已在 `backends/attention.py` 中实现。

### Trainer 类核心设计

```python
class Trainer:
    def __init__(self, cfg: dict): ...
    def train(self) -> None: ...
    def _save_checkpoint(self, step: int) -> None: ...
    def _log_metrics(self, step: int, loss: float, lr: float, epoch: int) -> None: ...
```

配置来自 YAML（由 `run.py` 用 `yaml.safe_load` 读取后传入），格式如 `configs/train.yaml` 结构。

**DDP 判断逻辑**：
```python
is_ddp = torch.distributed.is_available() and torch.distributed.is_initialized()
world_size = torch.distributed.get_world_size() if is_ddp else 1
```
- 单卡（`world_size == 1`）：直接使用 `AdamW`，无 `ShardedOptimizer`
- 多卡（`world_size > 1`）：用 `DDPIndividualParameters`（来自 Assignment2 ddp_training.py）包装模型，用 `ShardedOptimizer` 包装 `AdamW`

**metrics.jsonl 格式**（每步追加一行）：
```json
{"step": 1, "loss": 4.32, "lr": 3e-4, "epoch": 0}
```

**checkpoint 格式**（与 Plan 1 `save_checkpoint` 相同）：
```python
{"config": dataclasses.asdict(model_config), "state_dict": model.state_dict()}
```

**输出目录结构**：
```
results/{run_hash}/
  metrics.jsonl
  checkpoints/
    step_0100.pt
    step_0200.pt
    ...
```
`run_hash` 由 `hashlib.sha256(json.dumps(cfg, sort_keys=True, ensure_ascii=False).encode()).hexdigest()[:8]` 生成（与 `common/config.py` 的 `compute_cfg_hash()` 一致）。

```python
# ─────────────────────────────────────────
# Attention 后端注入辅助函数
# Purpose: 按 use_flash_attn 选择 attention 后端，通过猴子补丁注入各 TransformerBlock
# Key concept:
#   - use_flash_attn=True：委托 backends/attention.py 执行三级 fallback
#   - use_flash_attn=False：委托 backends/attention.py 短路返回 F.sdpa
#   - 猴子补丁：在 Transformer.__init__ 中替换各 attn.forward，保持原始类不变
# ─────────────────────────────────────────

def _build_attn_fn(use_flash_attn: bool):
    """委托 backends/attention.py 获取当前最优 attention 函数。"""
    from llm_foundry.backends.attention import get_attention_fn
    return get_attention_fn(use_flash_attn)


def _inject_attn_fn(transformer, attn_fn):
    """对 Transformer 的每个 TransformerBlock.attn 注入自定义 attention 函数。

    使用猴子补丁：替换 CausalMultiHeadSelfAttention.forward，
    用 attn_fn 替代原始的 F.scaled_dot_product_attention 调用。
    注入后在 block.attn 上设置 _attn_fn_injected=True 标记，便于测试验证。

    参数：
        transformer: Transformer 实例（BasicsTransformerLM 子类），含 self.layers
        attn_fn: callable，签名 (Q, K, V, is_causal=True) -> Tensor
    """
    from einops import rearrange
    import einx

    def _make_patched_forward(original_attn, fn):
        """工厂函数：为每个 attn 实例创建独立的 patched forward，避免闭包共享引用问题。"""
        def patched_forward(x, token_positions=None):
            *b, sequence_length, d_model = x.size()
            assert d_model == original_attn.d_model

            Q = original_attn.q_proj(x)
            K = original_attn.k_proj(x)
            V = original_attn.v_proj(x)

            Q, K, V = (
                rearrange(X, "... seq (heads d) -> ... heads seq d", heads=original_attn.num_heads)
                for X in (Q, K, V)
            )

            if token_positions is None:
                token_positions = einx.rearrange(
                    "seq -> b... seq",
                    torch.arange(sequence_length, device=x.device),
                    b=[1] * len(b)
                )

            token_positions = rearrange(token_positions, "... seq -> ... 1 seq")

            Q = original_attn.positional_encoder(Q, token_positions)
            K = original_attn.positional_encoder(K, token_positions)

            # 注入点：用 fn 替代原始的 F.scaled_dot_product_attention 调用
            attn_output = fn(Q, K, V, is_causal=True)

            attn_output = rearrange(attn_output, "batch heads seq d_v -> batch seq (heads d_v)").contiguous()
            output = original_attn.output_proj(attn_output)
            return output

        return patched_forward

    for block in transformer.layers:
        attn = block.attn
        attn.forward = _make_patched_forward(attn, attn_fn)
        attn._attn_fn_injected = True  # 注入标记，便于测试验证
```

然后修改 `Transformer.__init__` 末尾，在 `super().__init__(...)` 调用之后添加注入代码：

```python
"""
stage2_train/trainer.py — 主训练循环。

来源整合：
- 训练循环框架：reference_resource/Assignment1-basics/Transformer/train.py
- DDP 逻辑：reference_resource/Assignment2-system/cs336_systems/ddp_training.py
- 梯度累积/配置风格：reference_resource/Assignment4-data/cs336-basics/scripts/train.py

设计原则：
- 单卡（world_size=1）：直接使用 AdamW，无 DDP/ShardedOptimizer 包装
- 多卡（world_size>1）：DDPIndividualParameters + ShardedOptimizer(AdamW)
- 每步追加写入 metrics.jsonl，每 save_interval 步保存 checkpoint
- run_dir = {base_dir}/{run_hash}，hash 由配置 SHA256 前8位生成（可复现）
"""
from __future__ import annotations

import dataclasses
import json
import hashlib
import os
from typing import Any

import numpy as np
import torch
import torch.nn as nn
import torch.nn.functional as F

from llm_foundry.common.data import get_batch, load_tokens
from llm_foundry.common.model import ModelConfig, Transformer
from llm_foundry.common.optimizer import AdamW, ShardedOptimizer, get_cosine_lr


class Trainer:
    """LLM 训练器，支持单卡和 DDP+ZeRO-1 多卡训练。

    参数：
        cfg (dict): 完整配置字典，结构见 configs/train.yaml。
            必须包含 model / training / output 三个顶层键。

    属性：
        run_dir (str): 本次训练的输出目录，格式为 {base_dir}/{run_hash}。
    """

    def __init__(self, cfg: dict):
        self.cfg = cfg
        self.model_cfg_dict = cfg["model"]
        self.train_cfg = cfg["training"]
        self.out_cfg = cfg["output"]

        # 生成 run_hash：使用 json.dumps 保证与 common/config.py 的 compute_cfg_hash() 一致
        cfg_json = json.dumps(cfg, sort_keys=True, ensure_ascii=False)
        run_hash = hashlib.sha256(cfg_json.encode('utf-8')).hexdigest()[:8]
        self.run_dir = os.path.join(self.out_cfg["base_dir"], run_hash)
        self.ckpt_dir = os.path.join(self.run_dir, "checkpoints")
        self.metrics_path = os.path.join(self.run_dir, "metrics.jsonl")

    def train(self) -> None:
        """训练主循环。

        1. 初始化设备、模型、优化器、数据
        2. 判断单卡/多卡，按需启用 DDP + ZeRO-1
        3. 每步：forward -> cross_entropy -> backward -> clip grad -> step -> log -> save
        4. 结束时写入最终 checkpoint
        """
        # ── 设备初始化 ──
        device = self.train_cfg.get("device", "cpu")
        if device == "auto":
            device = (
                "cuda" if torch.cuda.is_available()
                else "mps" if torch.backends.mps.is_available()
                else "cpu"
            )

        # ── DDP 判断 ──
        # 检测是否在 torchrun 等分布式环境中（RANK 环境变量由 torchrun 自动注入）
        is_ddp = torch.distributed.is_available() and int(os.environ.get("RANK", -1)) != -1
        if is_ddp:
            # WSL 环境自动 fallback：优先 NCCL，失败则使用 gloo
            backend = "nccl" if torch.cuda.is_available() else "gloo"
            try:
                torch.distributed.init_process_group(backend=backend)
            except RuntimeError:
                print(f"[WARN] DDP backend '{backend}' 初始化失败，尝试 fallback 到 'gloo'")
                torch.distributed.init_process_group(backend="gloo")
            rank = int(os.environ["RANK"])
            local_rank = int(os.environ["LOCAL_RANK"])
            world_size = torch.distributed.get_world_size()
            device = f"cuda:{local_rank}"
            torch.cuda.set_device(device)
            is_master = rank == 0
        else:
            rank = 0
            world_size = 1
            is_master = True

        # ── 输出目录初始化（仅主进程创建） ──
        if is_master:
            os.makedirs(self.run_dir, exist_ok=True)
            os.makedirs(self.ckpt_dir, exist_ok=True)
            # 清空旧的 metrics.jsonl（若存在），保证重跑时 metrics 不累积
            if os.path.exists(self.metrics_path):
                os.remove(self.metrics_path)

        # ── 数据加载 ──
        data_path = self.train_cfg["data_path"]
        tokens = load_tokens(data_path)  # np.ndarray，内存映射模式

        # ── 模型初始化 ──
        model_config = ModelConfig(
            vocab_size=self.model_cfg_dict["vocab_size"],
            context_length=self.model_cfg_dict["context_length"],
            d_model=self.model_cfg_dict["d_model"],
            n_layers=self.model_cfg_dict["n_layers"],
            n_heads=self.model_cfg_dict["n_heads"],
            d_ff=self.model_cfg_dict["d_ff"],
            rope_theta=self.model_cfg_dict.get("rope_theta", 10000.0),
            use_flash_attn=self.model_cfg_dict.get("use_flash_attn", False),
        )
        model = Transformer(model_config).to(device)

        # ── DDP 包装（多卡） ──
        # 使用 Assignment2 的 DDPIndividualParameters（逐参数异步 AllReduce）
        if is_ddp and world_size > 1:
            from llm_foundry.common.ddp_utils import DDPIndividualParameters
            model = DDPIndividualParameters(model)

        # ── 优化器初始化 ──
        # 差异化权重衰减：2D+ 参数（矩阵）衰减，1D 参数（RMSNorm/偏置）不衰减
        param_dict = {pn: p for pn, p in model.named_parameters() if p.requires_grad}
        params_to_decay = [p for _, p in param_dict.items() if p.dim() >= 2]
        params_to_not_decay = [p for _, p in param_dict.items() if p.dim() < 2]
        optim_groups = [
            {"params": params_to_decay, "weight_decay": self.train_cfg.get("weight_decay", 0.1)},
            {"params": params_to_not_decay, "weight_decay": 0.0},
        ]

        if is_ddp and world_size > 1:
            # ZeRO-1：用 ShardedOptimizer 包装 AdamW，分片优化器状态
            optimizer = ShardedOptimizer(
                optim_groups,
                AdamW,
                lr=self.train_cfg["lr"],
                betas=(0.9, 0.999),
                eps=1e-8,
            )
        else:
            # 单卡：直接用 AdamW（避免 world_size=1 时 ShardedOptimizer 的额外复杂性）
            # 注意：weight_decay 已在 optim_groups 中按参数组指定，构造器不再传递
            optimizer = AdamW(
                optim_groups,
                lr=self.train_cfg["lr"],
                betas=(0.9, 0.999),
                eps=1e-8,
            )

        # ── 训练超参 ──
        max_steps = self.train_cfg["max_steps"]
        batch_size = self.train_cfg["batch_size"]
        context_length = model_config.context_length
        grad_accum_steps = self.train_cfg.get("gradient_accumulation_steps", 1)
        lr_max = self.train_cfg["lr"]
        lr_min = self.train_cfg.get("min_lr", lr_max * 0.1)
        warmup_steps = self.train_cfg.get("warmup_steps", max_steps // 20)
        grad_clip = self.train_cfg.get("grad_clip", 1.0)
        save_interval = self.train_cfg.get("save_interval", 500)
        log_interval = self.train_cfg.get("log_interval", 10)

        model.train()

        # ── 主训练循环 ──
        for step in range(1, max_steps + 1):
            # 余弦退火学习率调度（含线性预热）
            lr = get_cosine_lr(
                step,
                max_learning_rate=lr_max,
                min_learning_rate=lr_min,
                warmup_iters=warmup_steps,
                cosine_cycle_iters=max_steps,
            )
            for param_group in optimizer.param_groups:
                param_group["lr"] = lr

            # DDP 多卡路径禁用梯度累积：DDPIndividualParameters 每次 backward() 都触发 AllReduce，
            # 梯度累积时各 micro-step 各自同步，等价于对每个 micro-step 单独平均而非完整 batch 平均，语义错误。
            # 方案 B：强制 grad_accum_steps=1 消除歧义。
            if is_ddp and world_size > 1:
                if grad_accum_steps > 1:
                    print(f"[WARN] DDP 模式下 gradient_accumulation_steps={grad_accum_steps} 已强制设为 1")
                grad_accum_steps = 1
            total_loss = 0.0
            for micro_step in range(grad_accum_steps):

                x, y = get_batch(tokens, batch_size=batch_size,
                                 context_length=context_length, device=device)
                logits = model(x)
                # 交叉熵损失，除以梯度累积步数（等效于对小批次平均）
                loss = F.cross_entropy(
                    logits.view(-1, logits.size(-1)),
                    y.view(-1)
                ) / grad_accum_steps
                loss.backward()
                total_loss += loss.item()

            # DDP 多卡：等待所有异步 AllReduce 完成
            if is_ddp and world_size > 1 and hasattr(model, 'finish_gradient_synchronization'):
                model.finish_gradient_synchronization()

            # 梯度裁剪：防止梯度爆炸
            if grad_clip > 0:
                raw_model = model.module if hasattr(model, 'module') else model
                torch.nn.utils.clip_grad_norm_(raw_model.parameters(), grad_clip)

            optimizer.step()
            optimizer.zero_grad(set_to_none=True)

            # DDP 多卡（DDPBucketed）：重置桶就绪计数
            if is_ddp and world_size > 1 and hasattr(model, 'reset_buckets'):
                model.reset_buckets()

            # 计算 epoch（以 token 数估算）
            total_tokens_seen = step * batch_size * context_length * grad_accum_steps
            epoch = total_tokens_seen / max(len(tokens), 1)

            # ── 日志写入（每 log_interval 步，仅主进程） ──
            if is_master and step % log_interval == 0:
                actual_loss = total_loss  # total_loss 已是各 micro-step 的平均值（每步 loss/grad_accum_steps 后累加），无需乘回
                # 说明：每个 micro-step 的 loss 已除以 grad_accum_steps，total_loss 累加后等于整体平均
                # loss。乘回 grad_accum_steps 是错误的——会导致日志中 loss 虚高 grad_accum_steps 倍。
                self._log_metrics(step, actual_loss, lr, epoch)

            # ── Checkpoint 保存（每 save_interval 步，仅主进程） ──
            if is_master and step % save_interval == 0:
                raw_model = model.module if hasattr(model, 'module') else model
                # 若 DDP 包装，从 module 取出原始 Transformer
                if hasattr(raw_model, '_model_config'):
                    self._save_checkpoint(raw_model, step)
                else:
                    # 兼容：直接保存 state_dict（无 ModelConfig）
                    ckpt_path = os.path.join(self.ckpt_dir, f"step_{step:06d}.pt")
                    torch.save({"state_dict": raw_model.state_dict()}, ckpt_path)

        # ── 训练结束：保存最终 checkpoint（若最后一步未触发 save_interval） ──
        if is_master and max_steps % save_interval != 0:
            raw_model = model.module if hasattr(model, 'module') else model
            if hasattr(raw_model, '_model_config'):
                self._save_checkpoint(raw_model, max_steps)

        # ── DDP 清理 ──
        if is_ddp:
            torch.distributed.destroy_process_group()

    def _save_checkpoint(self, model: Transformer, step: int) -> None:
        """保存 Transformer checkpoint（兼容 from_checkpoint() 格式）。

        参数：
            model (Transformer): 含 _model_config 的 Transformer 实例（非 DDP 包装）
            step (int): 当前训练步数，用于文件名

        文件格式（与 Plan 1 save_checkpoint 格式一致）：
            {"config": dict, "state_dict": OrderedDict}
        """
        ckpt_path = os.path.join(self.ckpt_dir, f"step_{step:06d}.pt")
        torch.save(
            {
                "config": dataclasses.asdict(model._model_config),
                "state_dict": model.state_dict(),
            },
            ckpt_path,
        )

    def _log_metrics(self, step: int, loss: float, lr: float, epoch: float) -> None:
        """追加写入一行 metrics 到 metrics.jsonl。

        参数：
            step (int): 当前训练步数（从1开始）
            loss (float): 当前步的损失值
            lr (float): 当前步的学习率
            epoch (float): 当前步对应的 epoch 数（基于 token 数估算）

        输出格式（每行一个 JSON 对象）：
            {"step": 1, "loss": 4.32, "lr": 0.0003, "epoch": 0.0}
        """
        record = {
            "step": step,
            "loss": round(float(loss), 6),
            "lr": float(lr),
            "epoch": round(float(epoch), 4),
        }
        with open(self.metrics_path, "a", encoding="utf-8") as f:
            f.write(json.dumps(record) + "\n")
```

```python
def cmd_train(args) -> None:
    """实现 train 子命令：使用 load_config() 读取配置，调用 Trainer.train()。

    支持选项（均可在 YAML 或命令行参数中设置，命令行优先级更高）：
    - --config：YAML 配置文件路径（默认 configs/train.yaml）
    - --data：覆盖 training.data_path
    - --output：覆盖 output.base_dir
    - --flash-attn：强制启用 Flash Attention（覆盖 model.use_flash_attn）
    - --set：覆盖任意配置项（如 --set training.lr=1e-4）
    """
    from llm_foundry.stage2_train.trainer import Trainer
    from llm_foundry.common.env_check import check_env
    from llm_foundry.common.config import load_config

    # 使用 load_config 统一加载配置（支持 --set override，生成 config_snapshot.yaml）
    cfg, run_dir = load_config(args)
    cfg_dict = cfg.__dict__  # SimpleNamespace 转 dict

    # 命令行参数覆盖（--data, --output, --flash-attn）
    if args.data:
        cfg_dict.setdefault("training", {})["data_path"] = args.data
    if args.output:
        cfg_dict.setdefault("output", {})["base_dir"] = args.output
    if args.flash_attn:
        cfg_dict.setdefault("model", {})["use_flash_attn"] = True

    # 环境检测：若要求 Flash Attention 但无 GPU，给出警告（不 crash）
    env = check_env()
    if cfg_dict.get("model", {}).get("use_flash_attn", False) and not env.has_gpu:
        print("[WARN] use_flash_attn=True 但无可用 GPU，将自动 fallback 至 CPU SDPA")
        cfg_dict["model"]["use_flash_attn"] = False

    # DDP 检测：若 --ddp 但非 torchrun 环境，给出提示
    if getattr(args, 'ddp', False) and int(os.environ.get("RANK", -1)) == -1:
        print("[INFO] 检测到 --ddp，但当前非 torchrun 环境。")
        print("       多卡训练请使用：torchrun --nproc_per_node=N run.py train --config ...")
        print("       单卡训练继续执行...")

    print(f"[INFO] 开始训练，配置文件: {args.config or 'configs/train.yaml'}")
    trainer = Trainer(cfg_dict)
    print(f"[INFO] 输出目录: {trainer.run_dir}")
    trainer.train()
    print(f"[INFO] 训练完成。结果已写入: {trainer.run_dir}")
```
